# Brendan Nellis — Portfolio

This repository contains the source code for my personal portfolio website, built to showcase my background, projects, and technical skill set as an entry-level software engineer.

The site is implemented using plain HTML, CSS, and JavaScript to demonstrate a strong understanding of core web fundamentals without relying on frameworks.

I am a recent Electrical Engineering graduate with a concentration in Computer Engineering and a focus on Artificial Intelligence. My experience is project-driven and centered around building practical, end-to-end systems that combine frontend interfaces, backend-style logic, and real-world workflows.

This portfolio highlights my work in full-stack development, application logic, and systems-oriented problem solving.
